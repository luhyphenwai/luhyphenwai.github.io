UNITY PATHFINDING
What is this?

This is a visualization of a couple pathfinding algorithms which I implemented in Unity, like A*, Dijkstra, and Greedy BFS.

The green node is the starting position and the red node is the target position. Yellow nodes are already visited nodes and blue nodes are open nodes. You can use the draw tool to draw walls and move the starting and target position to test out the algorithm. The generate maze button uses a recursive maze generating algorithm.

If you set the pathfinding speed to the maximum, the pathfinding algorithm will instantly run, Otherwise, the algorithm will run the animations.
